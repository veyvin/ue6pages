<h2>Description / 描述</h2>
<div class="bilingual-block"><div class="lang-tabs"><div class="lang-tab active" onclick="switchLang(this)">中文</div><div class="lang-tab" onclick="switchLang(this)">English</div></div><div class="lang-content active">Unreal Path Tracer 的时空去噪引擎（主要与 MRQ 一起使用）。它根据空间和时间上各个方向上的周围斑块对每个像素进行去噪。该算法主要受到蒙特卡罗渲染去噪非线性加权一阶回归 (NFOR) 的启发。</div><div class="lang-content">Spatial-temporal denoising engine for the Unreal Path Tracer (mainly used with MRQ). It denoises each pixel based on the surrounding patches in space and time in all directions. The algorithm is mainly inspired by Nonlinearly Weighted First-order Regression (NFOR) for Denoising Monte Carlo Renderings.</div></div>
